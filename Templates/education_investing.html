Investing

Do I need to create an account to use this tool?

No! Creating an account allows this tool to be able to make smart calculations based on your goals and the data we receive. Doing so will also personalize your experience with this app. You can create an account here.



Is this tool free?

Yes! This tool is 100% free!



Who should use this tool?

This tool is designed especially for young adults between the ages of 18 and 25 to assist them on their path to financial literacy. It includes tips on saving and budgeting, investing, credit cards, loans, and more! Users are able to use as much or as little of these features as needed.



Do I need to have (financial) experience to use this tool?

Not at all! This tool will be customized to your financial literacy level -- whether you are a beginner, novice, or expert in this field! There are many resources for each user in differnt topics.
